# YouTube Audio Downloader This project is a Python-based audio downloader for YouTube videos that supports downloading Creative Commons content. ## Features - Download audio from YouTube videos - Support for Creative Commons content - Backend server setup ## Prerequisites - Python 3.x - Required libraries: `flask`, `youtube_dl` ## Installation 1. Clone the repository: ```bash git clone https://github.com/harukana2025/my-downloader.git cd my-downloader ``` 2. Install required packages: ```bash pip install Flask youtube_dl ``` ## Usage 1. Start the backend server: ```bash python app.py # Make sure to adjust according to your app script ``` 2. Use the web interface to input YouTube URLs and download audio files. ## Backend Server Setup ```python from flask import Flask, request, send_file import youtube_dl app = Flask(__name__) @app.route('/download', methods=['POST']) def download(): url = request.json['url'] ydl_opts = { 'format': 'bestaudio', 'postprocessors': [{ 'type': 'audiofile', 'format': 'mp3', }], } with youtube_dl.YoutubeDL(ydl_opts) as ydl: ydl.download([url]) # Add code to send the downloaded file back. if __name__ == '__main__': app.run(debug=True) ``` ## License This project is licensed under the MIT License.
